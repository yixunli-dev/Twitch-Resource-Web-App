# Twitch Resource Web App

A full-stack web application that allows users to search, browse, and favorite Twitch streaming content with a personalized experience.

---

## Overview

This project provides:

- Twitch content search (streams, videos, clips)
- Favorite system for saving content
- Basic recommendation based on user behavior
- Secure user authentication

---

## Tech Stack

**Backend**

- Java, Spring Boot
- Spring Security
- Hibernate / JPA
- MySQL

**Frontend**

- React
- Ant Design

**Infrastructure**

- Docker
- AWS ECR

---

## System Architecture
```
[ React Frontend ]
          ↓
[ Spring Boot Backend ]
          ↓
[ MySQL Database ]
```
---

## Project Structure

- backend/ - Spring Boot backend service
- frontend/ - React frontend application

---

## Getting Started

### Run Backend

cd backend
./gradlew bootRun

### Run Frontend

cd frontend
npm install
npm start
