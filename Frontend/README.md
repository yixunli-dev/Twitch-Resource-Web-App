# Frontend (React)

This is the client-side application for the Twitch Resource Web App. It provides the user interface and interacts with the backend REST APIs.

---

## Features

- Search Twitch content (streams, videos, clips)
- Display results in a responsive UI
- Save and manage favorite items
- User login and registration

---

## Tech Stack

- React
- Ant Design
- Axios (for API requests)

---

## Project Structure

```
components/ → reusable UI components
pages/ → main views
services/ → API communication
utils/ → helper functions
```


---

## How to Run
```
npm install
npm start
```

---

## Integration with Backend

- Communicates with Spring Boot backend via REST APIs
- Handles authentication and session management
- Fetches data dynamically from backend services
